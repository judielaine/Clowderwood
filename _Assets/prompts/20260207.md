# 2026 perennial purchases prompt

Following are three sets of purchase notes: the purchase order, followed by any vendor comments about the plants. For each purchased plants, other than the three execptions, generate a row in a csv file. You may investigate with https://plants.ces.ncsu.edu/ and https://www.missouribotanicalgarden.org for additional information to populate the fields. If you have access to a patent search you could look up patent information -- for example, 'Pardon My Lavender II'  is patented https://patents.google.com/patent/USPP34131P2/en  -- to inform the "terms" field

* Conradina canescens 'Gray Mound' (320)

Instruction 1: Usually height data is given as a range. Here, if the range is 5 to 10 ft, the values would be 7.5 with a delta of 2.5. Convert inches to feet.

Instruction 2:  Construct the note

If there is vendor provided information after the purchase text that is not included elsewhere in the entry, a brief factual sentence or two may be added here.

Instruction 3: For most plant families, the Header field is populated as follows

`Header: [Taxonomic family]: [Species] spp ([Species type])`

Established headers

* Asteraceae-Asteroideae-Anthemideae: Achillea spp
* Caprifoliaceae: Viburnum spp & var
* Lamiaceae-Nepetoideae-Mentheae: Clinopodium spp and hybrids
* Lamiaceae-Nepetoideae-Mentheae: Monarda spp
* Ericaceae: Rhododendron spp - native (azaleas and rhododendrons)

For the plants in family Liliaceae, use patterns like the following when taxonomy is contentious

* Liliaceae (alt Asphodelaceae)
* Liliaceae (alt Asphodelaceae)
* Liliaceae (alt Asparagaceae, Agavaceae)

For the plants in families Apiaceae, Asteraceae, Fabaceae, Lamiaceae, Poaceae, and Rosaceae use this as a pattern for the Header value

"Rosaceae-Amygdaloideae-Amygdaleae: Prunus spp (plums and cherries)"

Instruction 4:

If listed without a selection name but a scientific name

* Genetics: `Species`
* Terms:  `species`

If listed with a selection  and a scientific name (species given)
AND if patened or "PPAF" (plant patent applied for)

* Genetics: `Selection`
* Terms:   `patent` OR `PPAF`

AND if no patent information knwown

* Genetics: `Selection`
* Terms:  `unknown`

If listed with a selection  and only a genus or clearly a hybrid
AND if patened or "PPAF" (plant patent applied for)

* Genetics: `Hybrid`
* Terms:  `patent` OR `PPAF`

AND if no patent information knwown

* Genetics: `Hybrid`
* Terms:  `unknown`

Otherwise leave genetics blank and terms unknown

* Genetics: ""
* Terms:  `unknown`

Instruction 5:

Construct a comma separated list according to attached pdf list of terms for 'Bloom time'. 

If vendor or NCSU has an entry like "Summer, Late Summer/Early Fall" the value should be "summer, late summer, early autumn". That is, to express a range, use each period.
